gridoCSS

A simple, light and responsive 4KB FlexBox CSS Grid.

 
 
 
 
 
 
 
 
 
 

grido container

~ add 
.grido
class to your container 
<div class="grido">...</div>

grido items

~ add 
.gc_{size}
class to your items.
~ sizes: 2, 3, 4 ... up to 12
 
 
<div class="grido">
    <div class="gc_8">...</div>
    <div class="gc_4">...</div>
</div>

grido responsive

~ add 
.gc_{size}_{device}
class to your items.
~ devices: 
sm [mobile], md [tablet]

~ breakpoints: md 
max-width: 991px;
sm 
max-width: 767px;
 
 
<div class="grido">
    <div class="gc_4 gc_9_md gc_6_sm">...</div>
    <div class="gc_8 gc_3_md gc_6_sm"></div>
</div>

grido pads

~ add 
.grido--pads
class to the container to have padding between your items. 
<div class="grido grido--pads">...</div>
